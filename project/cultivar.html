O projeto trata-se de um pedido feito pela empresa de pulverizações por drones do sul de Minas Gerais — Cultivar Pulverizações — para a criação de um site estilo Landing Page, onde fosse possível além de constar todas as informações necessárias para levar o usuário a entrar em contato, também possuísse uma interface limpa, moderna e leve.

Para desenvolver a parte do UI Design, foi visado a criação de uma Landing Page limpa, moderna e leve. De forma a passar profissionalismo da empresa através do site. Na edição de imagens, foi utilizada a ferramenta Adobe Photoshop.

Para o desenvolvimento do front-end da página, foi utilizado as linguagens de marcação HTML5 e CSS3, juntamente com o JavaScript como linguagem de programação auxiliar.

Já para o back-end, foi utilizado o PHP para atuar no envio de email pelo formulário de contato.

Foi um projeto bem simples, realizado num período de basicamente 3 semanas, desde a criação do. Porém, que trouxe experiências únicas e aprendizados importantes para ambas as partes.
